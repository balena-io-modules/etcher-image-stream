# Change Log

All notable changes to this project will be documented in this file.
This project adheres to [Semantic Versioning](http://semver.org/).

## [v2.3.0] - 2016-07-01

### Added

- Add `hddimg` support.

## [v2.2.0] - 2016-06-21

### Added

- Add `dsk` support.

## [v2.1.0] - 2016-06-20

### Changed

- Set `zip` supported format type to `archive`.

## [v2.0.0] - 2016-05-19

### Added

- Add `gz` support.
- Add `bz2` support.

### Changed

- Extend `.supportedFileTypes` with metadata about the supported file types.

## [v1.2.0] - 2016-04-28

### Added

- Add support for Zip image directories.

## [v1.1.0] - 2016-04-27

### Added

- Export supported file types as `.supportedFileTypes`.

## [v1.0.1] - 2016-04-27

### Changed

- Don't ignore `lib/` in NPM package.
